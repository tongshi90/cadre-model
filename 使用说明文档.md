# 干部全量模型平台使用说明文档

  ## 一、平台简介

  干部全量模型平台是一款面向企业干部管理的综合信息系统，运用先进的数据分析算法，实现干部信息管理、岗位画像管理、干部岗位
  匹配度分析等核心功能，为组织人事决策提供科学依据，助力企业建立可持续发展的人才梯队。

---

  ## 二、平台核心能力

  ### 2.1 干部信息管理
  全面管理干部的基本信息、能力评分、特质分析、工作履历、培训记录、项目经验、绩效考核、奖惩情况等全量数据，建立完整的干部
  档案。

  ### 2.2 岗位画像管理
  维护企业各岗位的详细信息，包括岗位职责、编制数量、能力权重配置、任职要求等，构建精准的岗位画像。

  ### 2.3 智能匹配分析
  基于干部能力与岗位需求的智能匹配算法，自动计算匹配度评分，生成详细的分析报告，为干部选拔任用提供数据支撑。

  ### 2.4 可视化数据看板
  通过图表、雷达图等方式直观展示干部能力分布、匹配分析结果等关键信息，一目了然掌握人才队伍状况。

  ## 四、功能模块使用说明

  ### 4.1 干部管理

  #### 4.1.1 查看干部列表
  1. 点击顶部导航栏「干部管理」
  2. 系统以卡片视图展示所有干部信息
  3. 可通过部门筛选、姓名搜索快速定位目标干部

  **列表信息包含：**
  - 干部姓名、所属部门、现任岗位、在职状态
  - 快捷操作按钮（查看详情、编辑）

  #### 4.1.2 新增干部
  1. 在干部列表页面点击「新增干部」按钮
  2. 填写干部基本信息（姓名、性别、出生日期、联系方式、部门、岗位等）
  3. 点击「保存」完成创建

  #### 4.1.3 查看干部详情
  点击干部卡片进入详情页，包含以下信息模块：

  **【基本信息】** - 个人信息、联系方式、组织信息

  **【能力评估】** - 六大能力维度评分雷达图（领导能力、沟通协调、专业能力、学习创新、执行能力、团队管理）

  **【特质分析】** - 性格特质、管理风格、沟通风格、决策偏向

  **【干部履历】** - 培训记录、项目经验、绩效考核、奖惩情况、职务变更

  **【AI 分析】** - 智能分析报告和能力提升建议

  #### 4.1.4 编辑/删除干部
  - 点击「编辑」按钮修改信息
  - 点击「删除」按钮删除干部（需确认，不可恢复）

---

  ### 4.2 岗位管理

  #### 4.2.1 查看岗位列表
  系统以卡片视图展示所有岗位信息（岗位编码、名称、部门、编制、人数等）

  #### 4.2.2 新增岗位
  1. 点击「新增岗位」按钮
  2. 填写岗位基本信息（编码、名称、部门、编制、职责等）
  3. 配置能力权重（各能力维度的重要性占比，0-100）
  4. 设置岗位要求（硬性要求、建议要求）
  5. 点击「保存」完成创建

  #### 4.2.3 查看岗位详情
  包含基本信息、能力权重、岗位要求、任职干部等模块

---

  ### 4.3 部门管理

  #### 4.3.1 查看部门架构
  系统以树形结构展示组织架构，支持展开/收起

  #### 4.3.2 新增部门
  1. 点击「新增部门」按钮
  2. 选择上级部门（如为顶级部门则不选）
  3. 填写部门名称、编码、职责描述
  4. 点击「保存」完成创建

  #### 4.3.3 编辑/删除部门
  - 点击「编辑」修改部门信息
  - 点击「删除」删除部门（需确认无干部和子部门）

---

  ### 4.4 匹配分析

  匹配分析是平台的核心功能，用于评估干部与岗位的匹配程度。

  #### 4.4.1 分析操作步骤

  **方式一：当前岗位匹配分析**
  1. 点击顶部导航栏「匹配分析」
  2. 选择「当前岗位」标签页
  3. 选择目标岗位
  4. 选择分析对象（按部门选择或按干部选择）
  5. 点击「开始分析」按钮

  **方式二：自定义岗位匹配分析**
  1. 选择「自定义岗位」标签页
  2. 手动配置岗位能力权重和要求
  3. 选择分析对象
  4. 点击「开始分析」按钮

  #### 4.4.2 匹配结果解读

  | 字段 | 说明 |
  |------|------|
  | 干部姓名 | 被分析的干部 |
  | 目标岗位 | 分析的岗位 |
  | 匹配得分 | 匹配度评分（0-100 分） |
  | 匹配等级 | 优质匹配 / 合格匹配 / 不合格匹配 |
  | 操作 | 查看报告、查看干部详情 |

  **匹配等级说明：**
  - **优质匹配**：得分 ≥ 80 分，干部能力与岗位高度匹配
  - **合格匹配**：60 分 ≤ 得分 < 80 分，干部基本满足岗位要求
  - **不合格匹配**：得分 < 60 分，干部能力与岗位要求存在较大差距

  #### 4.4.3 查看匹配报告
  包含匹配概览、能力对比雷达图、要求检查、分析建议等内容

  ## 五、使用场景示例

  ### 场景 1：干部选拔任用
  为岗位选拔合适人选：进入「匹配分析」→ 选择岗位 → 选择干部 → 开始分析 → 查看结果和报告

  ### 场景 2：干部能力评估
  查看干部能力分布和短板：进入「干部管理」→ 找到干部 → 查看详情 → 查看能力评估和特质分析

  ### 场景 3：岗位编制分析
  分析人员配备情况：进入「岗位管理」→ 查看编制与实际人数 → 识别超编/缺编 → 制定调配方案

  ### 场景 4：新岗位画像设计
  设计新岗位画像：进入「岗位管理」→ 新增岗位 → 填写信息 → 配置权重和要求 → 保存验证

  ## 六、注意事项

  ### 6.1 数据维护
  - **及时更新**：干部信息发生变化时及时更新
  - **准确录入**：能力评分和特质标签应基于客观评价
  - **定期审核**：定期审核干部履历和岗位信息

  ### 6.2 匹配分析
  - **权重配置**：岗位能力权重应根据实际需求合理配置
  - **综合判断**：匹配度评分是参考依据，实际决策需结合多方面因素
  - **动态调整**：定期重新进行匹配分析

  ### 6.3 系统使用
  - **浏览器推荐**：建议使用 Chrome、Edge 等现代浏览器
  - **数据安全**：不要在公共场合展示敏感信息
  - **问题反馈**：遇到问题及时联系系统管理员

  ## 七、常见问题

  **Q1：为什么匹配分析结果显示「不合格匹配」？**
  A：可能是干部能力评分与岗位要求存在较大差距，或硬性要求未满足。建议查看详细报告了解具体原因。

  **Q2：可以批量导入干部信息吗？**
  A：当前版本暂不支持批量导入，需手动逐条录入。后续版本将增加该功能。

  **Q3：匹配分析的评分标准是什么？**
  A：匹配得分由基础得分和扣分组成。基础得分 = Σ(干部能力评分 ×
  岗位权重)，不满足硬性要求直接不合格，不满足建议要求会扣分。

  **Q4：如何修改岗位的能力权重配置？**
  A：进入岗位详情页，在「能力权重」模块点击「编辑」，调整各维度的权重值后保存。
